# ESTETIKA-WEB-SCRAPING

Welcome to the `ESTETIKA-WEB-SCRAPING` branch! This branch focuses on web scraping for the ESTETIKA platform. There are two Python files, namely `batik-shop.py` and `batik-tourist-experience.py`, designed for web scraping purposes to gather data related to Batik Shops and Batik Tourist Experiences.

## Files

- **batik-shop.py**: Python script for scraping data related to Batik Shops.
- **batik-tourist-experience.py**: Python script for scraping data related to Batik Tourist Experiences.

## Getting Started

To run the web scraping scripts locally, follow these steps:

1. Make sure you have Python installed. If not, download and install it from [https://www.python.org/](https://www.python.org/).

2. Clone the repository:

   ```bash
   git clone -b ESTETIKA-WEB-SCRAPING https://github.com/ESTETIKA-2023/ESTETIKA-CC.git
   ```

3. Navigate to the ESTETIKA-CC directory:

   ```bash
   cd ESTETIKA-CC
   ```

4. Create a virtual environment (optional but recommended):

    ```bash
    python -m venv .venv
    ```

    Activate the virtual environment:

    - On Windows:

        ```bash
        .venv\scripts\activate
        ```

    - On Unix or MacOS:

        ```bash
        source .venv/bin/activate
        ```

5. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

6. Run the web scraping script:
   - For Batik Shop data:

        ```bash
        python batik-shop.py
        ```
        
   - For Batik Tourist Experience data:

        ```bash
        python batik-tourist-experience.py
        ```
   The scripts will gather data from the web and save it in a structured format.
